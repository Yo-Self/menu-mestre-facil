# Guia Rápido: Horários de Funcionamento Automático

## 🚀 Como Usar (Para Proprietários de Restaurante)

### Passo 1: Acessar Configurações de Horários
1. Faça login no dashboard
2. Vá para "Restaurantes"
3. Clique no seu restaurante
4. Clique no botão **"Horários"** (ícone de relógio)

### Passo 2: Configurar Horários

Para cada dia da semana você pode:

#### Opção A: Restaurante ABERTO neste dia
1. ✅ Marque o switch como "Aberto"
2. ⏰ Defina o horário de **Abertura** (ex: 09:00)
3. ⏰ Defina o horário de **Fechamento** (ex: 22:00)

#### Opção B: Restaurante FECHADO neste dia
1. ❌ Marque o switch como "Fechado"
2. Os horários serão ignorados

### Passo 3: Atalho para Todos os Dias
Se todos os dias têm o mesmo horário:
1. Configure o primeiro dia
2. Clique em **"Copiar para todos"**
3. Todos os dias terão o mesmo horário

### Passo 4: Salvar
1. Clique em **"Salvar Horários"**
2. ✅ Pronto! O sistema agora funciona automaticamente

## 📋 Exemplos Práticos

### Exemplo 1: Restaurante Comercial
**Fecha aos domingos, aberto de segunda a sábado**

- **Domingo**: ❌ Fechado
- **Segunda a Sábado**: ✅ Aberto (11:00 - 22:00)

### Exemplo 2: Bar/Pub
**Fecha segundas, abre até tarde**

- **Segunda**: ❌ Fechado
- **Terça a Domingo**: ✅ Aberto (18:00 - 02:00)
  - O sistema entende que fecha às 02:00 do dia seguinte

### Exemplo 3: Padaria
**Abre cedo todos os dias**

- **Todos os dias**: ✅ Aberto (06:00 - 20:00)

## ⚡ Perguntas Frequentes

### Como funciona o fechamento automático?
- O sistema verifica a cada minuto se está no horário
- Quando chega no horário de abertura → Abre automaticamente
- Quando chega no horário de fechamento → Fecha automaticamente

### Posso ainda abrir/fechar manualmente?
**Sim!** O botão "Abrir/Fechar Restaurante" continua funcionando.
- Você pode abrir fora do horário se precisar
- Você pode fechar mais cedo se quiser
- Na próxima verificação automática, volta ao horário programado

### E se eu quiser mudar os horários?
Basta voltar na página de horários e alterar. As mudanças são aplicadas imediatamente.

### O que acontece se eu não configurar horários?
Nada! O restaurante continua funcionando normalmente, você só precisa abrir/fechar manualmente.

### Horários que passam da meia-noite funcionam?
**Sim!** Ex: 23:00 até 02:00
- O sistema automaticamente entende que 02:00 é do dia seguinte
- Funciona perfeitamente para bares, pubs, etc.

## ⚠️ Importante

1. **Timezone**: Os horários são baseados no horário do servidor
2. **Internet**: O sistema precisa estar online para funcionar
3. **Primeira vez**: Após configurar, aguarde até 1 minuto para primeira verificação

## 🔧 Troubleshooting

### O restaurante não abriu automaticamente
- ✅ Verifique se o horário está configurado
- ✅ Verifique se o dia está marcado como "Aberto"
- ✅ Aguarde até 1 minuto (verificação é a cada minuto)

### O restaurante não fechou automaticamente
- ✅ Verifique o horário de fechamento configurado
- ✅ Aguarde até 1 minuto

### Os horários não salvam
- ✅ Verifique se os horários estão no formato HH:MM
- ✅ Verifique se você tem permissão (é dono do restaurante)

## 💡 Dicas Profissionais

1. **Use "Copiar para todos"** para economizar tempo se a maioria dos dias têm o mesmo horário

2. **Configure com antecedência** - Os horários ficam salvos, você só precisa configurar uma vez

3. **Teste primeiro** - Configure para um dia e veja funcionando antes de aplicar para todos

4. **Horários flexíveis** - Você sempre pode abrir/fechar manualmente quando necessário

5. **Atualize quando mudar** - Se mudar os horários de funcionamento, lembre de atualizar aqui também
