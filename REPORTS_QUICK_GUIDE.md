# 🚀 Guia Rápido - Sistema de Relatórios

## Como Acessar

1. Faça login no Menu Mestre Fácil
2. Acesse o Dashboard
3. Clique em um restaurante
4. Clique no botão "Ver Relatórios" ou acesse:
   ```
   /dashboard/restaurants/[id]/reports
   ```

## 📊 O Que Você Vê

### Cards Principais
- **💵 Receita Total**: Quanto você faturou no período
- **📈 Ticket Médio**: Valor médio por pedido
- **📦 Item Mais Pedido**: Seu prato mais popular
- **🛒 Total de Pedidos**: Quantos pedidos foram feitos

### Análises
- **Top 5 Itens**: Seus best-sellers com quantidade e receita
- **Horários de Pico**: Quando você tem mais movimento

### Lista de Pedidos
Todos os pedidos com detalhes completos:
- Status colorido
- Cliente e mesa
- Itens pedidos
- Valores

## ⏰ Períodos Disponíveis

### Diário
- Pedidos de hoje (00:00 até agora)
- Útil para acompanhamento em tempo real

### Semanal
- Pedidos da semana atual (domingo a hoje)
- Bom para planejamento semanal

### Mensal
- Todos os pedidos do mês
- Ideal para fechamento mensal

### Personalizado
1. Clique na aba "Personalizado"
2. Escolha a data inicial
3. Escolha a data final
4. Clique em "Gerar Relatório"

**Dica:** Use para comparar feriados, fins de semana, etc.

## 💾 Como Exportar

1. Configure o período desejado
2. Clique em "Exportar CSV" (canto superior direito)
3. O arquivo será baixado automaticamente
4. Abra no Excel ou Google Sheets

### O Que Vem no CSV?
- ID do pedido
- Data e hora
- Nome do cliente
- Mesa
- Status
- Cada item pedido (nome, quantidade, preço)
- Total do pedido

## 📈 Dicas de Uso

### Para Aumentar Vendas
1. Veja o **Item Mais Pedido** e crie combos
2. Analise **Horários de Pico** e ajuste equipe
3. Use o **Ticket Médio** como meta de vendas

### Para Controle Financeiro
1. Compare **Receita Total** entre períodos
2. Monitore pedidos **Cancelados**
3. Analise evolução do **Ticket Médio**

### Para Gestão de Cardápio
1. Identifique itens no **Top 5**
2. Avalie se itens fora do ranking devem continuar
3. Use dados para criar promoções estratégicas

### Para Operação
1. Use **Horários de Pico** para escalar equipe
2. Prepare ingredientes baseado em itens populares
3. Monitore status dos pedidos em tempo real

## 🎯 Interpretação dos Dados

### Ticket Médio Alto
- ✅ Clientes gastam mais
- 💡 Oportunidade para programa de fidelidade
- 💡 Investir em itens premium

### Ticket Médio Baixo
- 🤔 Clientes gastam menos
- 💡 Criar combos promocionais
- 💡 Implementar sugestões de complementos

### Taxa de Cancelamento Alta
- ⚠️ Pode indicar problemas
- 💡 Investigar motivos (tempo de espera, qualidade)
- 💡 Melhorar comunicação com cliente

### Horários de Pico Concentrados
- 📊 Demanda concentrada
- 💡 Oferecer desconto em horários vazios
- 💡 Preparar mais equipe no pico

## ❓ Perguntas Frequentes

### Não vejo nenhum pedido
- Verifique se o período está correto
- Confirme se há pedidos para este restaurante
- Tente outro período

### Valores parecem errados
- Lembre-se: apenas pedidos **finalizados** contam na receita
- Pedidos **cancelados** aparecem mas não somam na receita
- Verifique o status de cada pedido

### Exportação não funciona
- Verifique se há pedidos no período
- Tente em outro navegador
- Certifique-se que downloads estão permitidos

### Item aparece como "Item removido"
- O prato foi deletado do cardápio
- O pedido está preservado com preço histórico
- Normal para itens antigos

## 🔄 Atualizações

O relatório é atualizado:
- ✅ Ao trocar de período
- ✅ Ao gerar relatório personalizado
- ✅ Sempre que você acessar a página

**Nota:** Para ver dados mais recentes, simplesmente recarregue a página ou troque de aba.

## 💡 Casos de Uso

### Exemplo 1: Preparar para o Fim de Semana
1. Acesse o relatório **Semanal**
2. Veja os **Horários de Pico**
3. Identifique o **Top 5 Itens**
4. Prepare estoque extra desses itens
5. Escale equipe para os horários de pico

### Exemplo 2: Análise Mensal
1. Acesse o relatório **Mensal**
2. Exporte o **CSV**
3. Compare com mês anterior
4. Identifique tendências
5. Ajuste estratégia para próximo mês

### Exemplo 3: Avaliar Promoção
1. Use período **Personalizado**
2. Selecione dias da promoção
3. Compare **Receita** e **Ticket Médio**
4. Veja se item promocional entrou no **Top 5**
5. Calcule ROI da promoção
